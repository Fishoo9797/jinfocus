# 快速上手 · 从零部署「家人日记」

预计用时：**20 分钟**（含下载工具）。无需服务器、无需备案。

## 准备

1. 一个微信账号（个人号即可）。
2. 电脑安装 **[微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)**（稳定版）。

---

## 第 1 步：注册小程序，拿到 AppID

1. 打开 [mp.weixin.qq.com](https://mp.weixin.qq.com/) → 立即注册 → 选择「小程序」。
2. 用邮箱注册，激活后登录。
3. 进入 **开发 → 开发管理 → 开发设置**，复制 **AppID（小程序ID）**。

> 个人主体即可，无需企业资质；云开发个人版免费额度足够家庭使用。

---

## 第 2 步：导入项目

1. 打开微信开发者工具 → 「导入项目」。
2. 目录选择 `journal-miniapp/` 文件夹（即本仓库根目录）。
3. AppID 填第 1 步拿到的；项目名称随意。
4. 导入后，把 `project.config.json` 里的 `"appid": "YOUR_APPID"` 改成你的真实 AppID 并保存。

---

## 第 3 步：开通云开发，拿到环境 ID

1. 开发者工具顶部点 **「云开发」** 按钮 → 开通。
2. 创建环境（名字随意，如 `family-journal-env`）。
3. 开通后在「云开发控制台 → 设置 → 环境」复制 **环境 ID**。
4. 打开 `miniprogram/app.js`，把：
   ```js
   envId: 'YOUR_ENV_ID',
   ```
   改成你的环境 ID，保存。

---

## 第 4 步：创建数据库集合

进入「云开发控制台 → 数据库 → 新建集合」，依次创建三个集合：

- `users`
- `families`
- `journals`

（权限先保持默认即可，因为数据读写都走云函数，不受端侧权限影响。）

---

## 第 5 步：上传并部署云函数

在左侧文件树 `cloudfunctions/` 下，**对每个云函数文件夹**分别操作：

1. 右键该文件夹（如 `login`）→ 「在终端中打开」。
2. 执行 `npm install`（会按 `package.json` 装 `wx-server-sdk`）。
3. 回到开发者工具，右键该文件夹 → **「上传并部署：云端安装依赖」**。

需要对以下 6 个都做一遍：
`login` / `saveJournal` / `getJournals` / `getFamily` / `createFamily` / `joinFamily`

---

## 第 6 步：真机预览 & 邀请家人

1. 点开发者工具的 **「预览」**，用手机微信扫码打开。
2. 在「我的」页 → **创建家庭**，家庭名随意（如「盈的一家」）。
3. 把生成的**家庭 ID** 复制发给家人。
4. 家人在自己手机打开同一小程序 → 「我的」→ **加入家庭**，粘贴 ID 即可。
5. 之后大家每天在「今日」写三问，去「家人」互看。

---

## 常见问题

**Q：家人互相看不到日记？**
A：确认都在同一个家庭 ID 下（各自「我的」页显示的 familyName 一致）。

**Q：语音播放没声音？**
A：手机需允许小程序使用麦克风（首次录音时弹窗授权）；试听走的是云存储下载，需联网。

**Q：想改三个问题？**
A：改 `miniprogram/pages/index/index.js` 里的 `QUESTIONS` 常量，重新编译即可。

**Q：云函数报 `DYNAMIC_CURRENT_ENV` 相关错误？**
A：确认第 3 步环境 ID 已正确填进 `app.js`，且 6 个云函数均已「上传并部署」。

**Q：数据会丢吗？**
A：云开发有自动备份；家庭项目量级极小，免费额度充裕。
